# ssllabs-scan
Python script to automate security scanning of websites using Qualys SSL Server Testing

# Usage
# Linux
1) virtualenv env
2) . env/bin/activate
3) pip install -e .
4) ssllabs-scan sample_website.txt
# Windows
1) virtualenv env
2) env\Scripts\activate
3) pip install -e .
4) ssllabs-scan sample_website.txt
